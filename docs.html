Command Documentation

Here’s a comprehensive list of commands for CBRN Bot:

!hazmat - Provides guidelines for handling hazardous materials.
!alert - Issues an emergency alert to all channels.
!safety [type] - Safety advice based on CBRN type (e.g., chemical, biological).
!evacuate - Provides evacuation procedures for CBRN incidents.
!status - Displays bot’s operational status and current alerts.
!decontaminate - Guidelines for decontamination procedures.
!report [incident] - Logs and reports a new CBRN incident.
